We have used an open source data set — Face Emotion Recognition (FER) from Kaggle and built a CNN to detect emotions. The emotions can be classified into 7 classes — happy, sad, fear, disgust, angry, neutral and surprise. Model — We built a 6 layered Convolutional Neural Network (CNN) in Keras and use image augmentations to improve model performance.
